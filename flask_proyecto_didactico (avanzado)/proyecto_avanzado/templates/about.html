Arquitectura

  • App factory: `create_app` en `__init__.py`.
  • Blueprints: `routes/web.py` para HTML, `routes/api.py` para JSON.
  • Controladores/servicios: `controllers/items.py` maneja logica.
  • Modelos simples: `models.py` con dataclass Item.

Practicas sugeridas

  1. Agrupa rutas nuevas en otros blueprints (auth, admin).
  2. Persistencia: cambia el servicio a SQLite o SQLAlchemy.
  3. Agrega tests para la API con pytest y client de Flask.
  4. Mejora UX: flash messages y validaciones en formularios.
