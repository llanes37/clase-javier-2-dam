Acerca de esta aplicacion

Objetivo

Aprender Flask desde cero mediante ejemplos practicos en un proyecto pequeno.

Caracteristicas
  • Plantillas Jinja2: Guardadas en /templates con herencia.
  • Rutas con parametros: Tipado automatico (int, str).
  • Formularios POST: request.form para recibir datos.
  • Query strings GET: request.args para parametros en URL.
  • API JSON: endpoints que devuelven JSON con codigos HTTP.
  • Manejadores de errores: @app.errorhandler(404), (500).
  • Hooks (before/after): Se ejecutan antes y despues de cada peticion.
Estructura de codigo
1. Inicializacion de la app
2. Rutas (endpoints)
3. Mini API JSON
4. Manejo de errores
5. Hooks before/after
6. Punto de entrada (if __name__ == "__main__")
Proximos pasos
  • Lee el archivo app.py comentado paso a paso.
  • Modifica las rutas y plantillas.
  • Anade nuevas rutas y funciones.
  • Conecta una base de datos SQLite o PostgreSQL.
  • Despliega en Render, Fly.io o PythonAnywhere.
